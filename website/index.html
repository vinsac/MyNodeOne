Get Started in 3 Steps

1

Run Interactive Setup

One command sets up everything. Works on any machine with Ubuntu 24.04.

git clone https://github.com/vinsac/MyNodeOne.git cd mynodeone sudo ./scripts/mynodeone
2

Bootstrap Your Cluster

Installs Kubernetes, storage, monitoring, and GitOps in ~20 minutes.

sudo ./scripts/bootstrap-control-plane.sh
3

Deploy Your Apps

Push to Git and ArgoCD automatically deploys. Or use our examples.

./scripts/create-app.sh myapp --domain myapp.com kubectl apply -f manifests/examples/fullstack-app.yaml
